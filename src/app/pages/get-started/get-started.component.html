Get Started
This UI Kit is designed for the Landing Pages that will be made from now on. Here you can find a variety of components that will allow you to speed up the process of making them.
How to use it
To use it, you must download the folder of the . Then paste it on your project, at the src folder (at the same label of assets) . Go to "app.module.ts" and import it. {{ "import { UiKitModule } from 'src/shared/ui-kit.module'; " }} Then add it to imports on NgModule @NgModule ({
    declarations: [],
    imports: [
-->      UiKitModule
     ],
}) Finally, go to angular.json, and paste "src/shared/ui-kit/icons" on assets (there are 2)    "src/favicon.ico",
   "src/assets",
--> "src/shared/ui-kit/icons"
 Now it's ready to use!
